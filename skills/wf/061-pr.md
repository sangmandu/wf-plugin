## Checklist

- [ ] Push branch: `git push -u origin <TICKET-ID>`
- [ ] `gh pr create` (NOT `--draft`):
  - [ ] Title: `type(scope): [TICKET-ID] Short description`
  - [ ] Label: auto-determine per draft-pr label rules (if available)
  - [ ] Assignee: `@me`
  - [ ] Base branch: `main`
  - [ ] Body: **must follow the template below**
- [ ] Save `pr_number` and `pr_url` to state.json
- [ ] **Verify PR is open and NOT draft**: `gh pr view <pr_number> --json isDraft,state --jq '.'` must show `isDraft: false` and `state: OPEN`.
